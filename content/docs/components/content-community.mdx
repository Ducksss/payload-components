## Content model

The `eyebrow`, `title`, and `paragraphs` fields come from the shared Content family base
(`src/blocks/shared/contentFields.ts`); the `avatars` list is specific to this variant.

<TypeTable
  type={{
    eyebrow: { type: 'text', description: 'Optional small label shown above the centered title.' },
    title: { type: 'text', required: true, description: 'The section heading.' },
    paragraphs: {
      type: 'array',
      description: 'Centered body copy — one or more short paragraphs (min 1, max 4).',
    },
    avatars: {
      type: 'array',
      required: true,
      description: 'The wall of community members (min 3, max 24).',
    },
  }}
/>

Each item in `avatars` carries:

<TypeTable
  type={{
    avatar: { type: 'upload', required: true, description: 'Avatar image (Media upload).' },
    name: { type: 'text', required: true, description: 'Member name (used as the link title).' },
    href: { type: 'text', description: 'Optional link to the member’s profile.' },
  }}
/>
