## Content model

This serif "list" variant uses its own fields (not the shared `contentFields` base): a heading and
intro above a multi-column icon list. Item icons are chosen from the shared allowlist in
`src/blocks/shared/contentIcons.ts`.

<TypeTable
  type={{
    eyebrow: { type: 'text', description: 'Optional small label shown above the title.' },
    title: { type: 'text', required: true, description: 'The serif section heading.' },
    description: { type: 'textarea', description: 'Optional intro copy under the heading.' },
    items: {
      type: 'array',
      required: true,
      description: 'The icon list (min 2, max 6), shown in a multi-column grid.',
    },
  }}
/>

Each item in `items` carries:

<TypeTable
  type={{
    icon: {
      type: 'select',
      description: 'Lucide icon name (zap · cpu · lock · sparkles · gauge · shield).',
    },
    term: { type: 'text', required: true, description: 'The lead-in term (rendered emphasized).' },
    description: { type: 'textarea', required: true, description: 'The term’s description.' },
  }}
/>
