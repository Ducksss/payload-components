## Content model

`pricingFields` carries the section header and the repeatable plan cards. This block is presentation
only. It does not add checkout, subscriptions, entitlements, customer portals, or billing logic.

<TypeTable
  type={{
    eyebrow: { type: 'text', description: 'Small label shown above the title.' },
    title: { type: 'text', required: true, description: 'The pricing section heading.' },
    description: { type: 'textarea', description: 'Optional supporting copy under the heading.' },
    plans: {
      type: 'array',
      required: true,
      description: 'Two to four plan cards. One plan can be highlighted as the recommended tier.',
    },
  }}
/>

Each item in `plans` carries:

<TypeTable
  type={{
    name: { type: 'text', required: true, description: 'Plan name.' },
    description: { type: 'textarea', description: 'Short plan summary.' },
    price: { type: 'text', required: true, description: 'Price label, such as "$29" or "Custom".' },
    billingNote: { type: 'text', description: 'Small note beside the price, such as "per month".' },
    highlighted: { type: 'checkbox', description: 'Emphasizes the card as the recommended tier.' },
    features: {
      type: 'array',
      required: true,
      description: 'Two to eight feature bullets. Long bullets wrap inside the card.',
    },
    links: { type: 'array', description: 'One optional CTA link for the plan.' },
  }}
/>

Each item in `features` carries:

<TypeTable
  type={{
    text: { type: 'text', required: true, description: 'Feature bullet text.' },
  }}
/>
