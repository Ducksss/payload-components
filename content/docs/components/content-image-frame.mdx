## Content model

The `eyebrow`, `title`, and `paragraphs` fields come from the shared Content family base
(`src/blocks/shared/contentFields.ts`); the `image` and optional `backgroundImage` uploads are
specific to this variant. The image is framed inside a rounded backdrop.

<TypeTable
  type={{
    eyebrow: { type: 'text', description: 'Optional small label shown above the centered title.' },
    title: { type: 'text', required: true, description: 'The section heading.' },
    paragraphs: {
      type: 'array',
      description: 'Centered intro copy — one or more short paragraphs (min 1, max 4).',
    },
    image: { type: 'upload', description: 'Foreground screenshot (Media upload) shown in the frame.' },
    backgroundImage: { type: 'upload', description: 'Optional backdrop image (Media upload) behind the frame.' },
  }}
/>
