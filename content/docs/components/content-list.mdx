## Content model

This serif "list" variant uses its own fields (not the shared `contentFields` base): a heading beside
a list of labeled-term paragraphs.

<TypeTable
  type={{
    eyebrow: { type: 'text', description: 'Optional small label shown above the title.' },
    title: { type: 'text', required: true, description: 'The serif section heading.' },
    items: {
      type: 'array',
      required: true,
      description: 'The labeled-term list (min 2, max 6).',
    },
  }}
/>

Each item in `items` carries:

<TypeTable
  type={{
    term: { type: 'text', required: true, description: 'The lead-in term (rendered emphasized).' },
    description: { type: 'textarea', required: true, description: 'The term’s description.' },
  }}
/>
