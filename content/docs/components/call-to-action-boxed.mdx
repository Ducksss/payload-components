## Content model

`title` and `description` come from the shared `callToActionFields` base; `links` is specific to this
variant. Same fields as `call-to-action-centered` — this variant nests the content in a bordered panel
for higher emphasis.

<TypeTable
  type={{
    title: { type: 'text', required: true, description: 'The call-to-action heading.' },
    description: { type: 'textarea', description: 'Optional supporting copy under the heading.' },
    links: { type: 'array', description: 'One or two CTA links (primary + optional secondary).' },
  }}
/>
