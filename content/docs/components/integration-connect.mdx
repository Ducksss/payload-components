## Content model

`heading`, `subtext`, and `integrations` come from the shared `integrationFields` base. This variant
adds an optional `featuredLogo` the connector lines route to.

<TypeTable
  type={{
    heading: {
      type: 'text',
      required: true,
      description: 'Short heading shown under the diagram.',
    },
    subtext: {
      type: 'textarea',
      description: 'Optional supporting paragraph under the heading.',
    },
    integrations: {
      type: 'array',
      required: true,
      description: 'Two to twelve integrations, each an uploaded logo with a name.',
    },
    featuredLogo: {
      type: 'upload',
      description: 'Optional center brand mark the connector lines route to.',
    },
  }}
/>

Each item in `integrations` carries:

<TypeTable
  type={{
    logo: { type: 'upload', required: true, description: 'Logo image (Media upload).' },
    name: { type: 'text', required: true, description: 'Accessible name / alt text for the logo.' },
    description: { type: 'textarea', description: 'Optional blurb (unused by this variant).' },
    href: { type: 'text', description: 'Optional link (unused by this variant).' },
  }}
/>
