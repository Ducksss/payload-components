## Content model

The `eyebrow` and `title` fields come from the shared Team family base
(`src/blocks/shared/teamFields.ts`); the `groups` list is specific to this variant.

<TypeTable
  type={{
    eyebrow: { type: 'text', description: 'Optional small label shown above the title.' },
    title: { type: 'text', required: true, description: 'The section heading.' },
    groups: {
      type: 'array',
      required: true,
      description: 'Department sections, each with its own label and members (min 1, max 6).',
    },
  }}
/>

Each item in `groups` carries:

<TypeTable
  type={{
    label: { type: 'text', required: true, description: 'Department / group heading (e.g. “Leadership”).' },
    members: {
      type: 'array',
      required: true,
      description: 'The members in this group (min 1, max 12).',
    },
  }}
/>

Each item in `members` carries:

<TypeTable
  type={{
    avatar: { type: 'upload', required: true, description: 'Member photo (Media upload).' },
    name: { type: 'text', required: true, description: 'Member name.' },
    role: { type: 'text', required: true, description: 'Job title or role.' },
    href: { type: 'text', description: 'Optional link to the member’s profile.' },
  }}
/>
