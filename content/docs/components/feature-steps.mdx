## Content model

The first three fields come from the shared `featureFields` base; `items` and `links` are specific
to this variant.

  type={{

    description: { type: 'textarea', description: 'Optional intro copy.' },
    items: {
      type: 'array',
      required: true,
      description: 'Two to six steps, numbered automatically in array order.',
    },
    links: { type: 'array', description: 'One optional CTA link.' },
  }}
